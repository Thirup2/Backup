## 一. 适用情况
希望从表达式的类型推断出要定义的变量的类型，但是不想用该表达式的值初始化变量。

## 二. 使用方法
```c++
decltype(expression) name = value;
```
decltype根据表达式选择并返回操作数的类型。
编译器分析表达式并得到它的类型，但是不实际计算表达式的值

## 三. 特殊情况
### 1. const和引用
如果表达式是一个变量，则decltype返回该变量的类型（包括顶层const和引用在内）：
```c++
const int ci = 0, &cj = ci;
decltype(ci) x = 0;        // x的类型是const int
decltype(cj) y = x;        // y的类型是const int &
```
此处并未把引用当成其所指的对象进行分析

### 2. decltype和引用
如果decltype使用的表达式不是一个变量，则decltype返回表达式结果对应的类型。
**当表达式的结果对象能作为一条赋值语句的左值时，将返回一个引用类型**：
```c++
int i = 42;
int *p = &i;
int &r = i;
decltype(r) a = i;        // r是一个整型引用，所以a是一个整型引用，必须进行初始化
decltype(r+0) b;          // r+0的结果是一个整型，所以b是一个整型变量
decltype(*p) c;           // 错误：*p是一个可作为左值的表达式，所以返回整型引用类型，必须初始化
```
**注意**：**如果decltype所用的表达式是变量名加上了一对括号，则结果将永远是引用**。因为加上了括号的变量是一个表达式，而表达式的结果是一个可作为左值的对象。
